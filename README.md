# Fdf

This is the first 42 graphic project.

Using MinilibX.

# Getting started

### Clone the repo :
```
git clone https://github.com/VirgileVP/Fdf
```

### Install :
```
make
```

### Exec :
```
./Fdf <your_map>
```

### How to use :

Horizontal and vertical moves : WASD

Diagonal moves : QEZC

Moving speed : + -

Change colors : R-> 1 4   G-> 2 5   B-> 3 6

Change height : R F

Invert height : I

Change Projection : P
